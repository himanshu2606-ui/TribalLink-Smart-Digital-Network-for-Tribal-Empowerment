TribalLink – Smart Digital Network for Tribal Empowerment
----------------------------------------------------------

This is a student-level demo project built by Team TechTribe.

Project Overview:
-----------------
TribalLink is a digital platform designed to connect and empower tribal (Adivasi) communities
through technology. The platform provides a space for artisans, farmers, and entrepreneurs
to showcase their products, access learning resources, and communicate with others.

Folder Structure:
-----------------
demo/
 ┣━ app.py                 → Main Flask backend file
 ┣━ static/                → Contains all CSS, JS, and image files
 ┣━ templates/             → Contains HTML templates
 ┣━ CREDITS.txt            → Team and contribution details
 ┣━ README.txt             → Project documentation (this file)
 ┗━ run_demo.sh            → Shell script to run the demo

How to Run the Demo:
--------------------
1. Make sure Python 3.x is installed.
2. Open a terminal inside the 'demo' folder.
3. Install Flask by running:
       pip install flask
4. Run the demo app:
       python app.py
5. Open your browser and visit:
       http://127.0.0.1:5000/
