TROUBLESHOOTING
================================================================================

‚ùå Problem: "Python is not recognized"
   ‚úì Solution: Make sure Python is in PATH
   ‚Ä¢ Go to System Environment Variables
   ‚Ä¢ Add Python installation path (usually C:\Python39\)

‚ùå Problem: "Flask not found"
   ‚úì Solution: Run this command:
     pip install flask flask-cors

‚ùå Problem: "Address already in use" (Port 5000)
   ‚úì Solution: Change port in app.py
   ‚Ä¢ Open app.py in editor
   ‚Ä¢ Find: app.run(debug=True, host='127.0.0.1', port=5000)
   ‚Ä¢ Change 5000 to 5001: port=5001
   ‚Ä¢ Run again: python app.py
   ‚Ä¢ Visit: http://127.0.0.1:5001

‚ùå Problem: "Static files not loading" (no CSS/JS)
   ‚úì Solution:
   ‚Ä¢ Hard refresh: Ctrl+Shift+Delete
   ‚Ä¢ Make sure static/ folder exists
   ‚Ä¢ Check file names are correct

‚ùå Problem: Chatbot not responding
   ‚úì Solution:
   ‚Ä¢ Check internet connection
   ‚Ä¢ Look at browser console (F12)
   ‚Ä¢ Make sure Flask server is running

      ‚Ä¢ Look for IPv4 Address (like 192.168.1.100)

      (Replace 192.168.1.100 with your actual IP)

OPTION B: Share GitHub Repository
   https://github.com/himanshu2606-ui/TribalLink-Smart-Digital-Network-for-Tribal-Empowerment

OPTION C: Create Public Link with Ngrok (Free)
   1. Install ngrok:
      pip install pyngrok
   
   2. Run ngrok:
      python -m pyngrok.ngrok http 5000
   
   3. Copy the link provided (e.g., https://xxxx-xxx.ngrok.io)
   4. Share with team!

FREQUENTLY ASKED QUESTIONS
================================================================================

Q: Can I run this on my phone?
A: Yes! Use the local network link (http://192.168.x.x:5000) from any device on same WiFi

Q: Can I share this with people outside my network?
A: Use Ngrok for free public link, or deploy to cloud (Heroku, AWS, etc.)

Q: How do I edit the products list?
A: Edit the PRODUCTS array in app.py (around line 20)

Q: Can I add a database?
A: Yes! Replace PRODUCTS list with SQLite, MySQL, or PostgreSQL

Q: How do I deploy this online?
A: Use services like Heroku, AWS, Google Cloud, or PythonAnywhere

Q: What if I want to add user authentication?
A: Install Flask-Login and create a database with user credentials

Q: Can I use this as a starter template?
A: Absolutely! Feel free to fork and customize for your needs
